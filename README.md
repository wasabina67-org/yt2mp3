# yt2mp3
YouTube to MP3

## Setup

```bash
cp -p src/data.py.example src/data.py
```

## Run

```bash
python src/run.py
```

```bash
ls output/*.mp3
```

```bash
ls output/*.mp3 | wc -l
```
